License Optimization

Reduce or increase allocations to align cost with actual workload demand.

Standard Licenses

Reduce: Move {{ azure_payg_count }} PAYG candidates to Azure Pay-As-You-Go to reduce standard license count and cost.

Increase: Add only where production workloads genuinely require full standard coverage.

Target: align standard count with actual production demand.

Developer Licenses

Increase: Use developer editions for dev and test instead of standard or enterprise to cut cost.

Reduce: Remove developer licenses for retired or unused environments such as the {{ retired_count }} flagged devices.

Target: dev and test on developer only, never production.

Enterprise Licenses

Reduce: Migrate eligible workloads to PAYG or Standard where enterprise-only features are not required.

Increase: Reserve enterprise for systems that need advanced capabilities and consolidate duplicate coverage.

Target: right-size enterprise demand against {{ total_demand_quantity }} total required licenses.
